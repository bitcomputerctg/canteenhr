How device sync works
Each ZKTeco machine pushes attendance punches and canteen meal scans straight into Firestore via the bridge script — new records appear here automatically within seconds, no manual action needed. Use Sync Now on a device to pull the latest records immediately (useful right after connecting a new machine, or to confirm a device is reachable). If the backend is unavailable, the dashboard falls back to local demo data and re-syncs automatically once the connection is restored.
